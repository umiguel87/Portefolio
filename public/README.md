# Ficheiros na pasta public

Coloca aqui os ficheiros que o portefólio usa por URL:

- **cv.pdf** — CV para o botão "Descarregar CV" (pode ser uma cópia do teu `CV_Luis_Rodrigues.pdf`)
- **br_vr.jpeg** — Logo BR&VR (Estágios)
- **core.png** — Logo Core Soluções (Estágios)
- **ipvc.jpg** — Logo IPVC (Escolaridade)
- **sanches.jpeg** — Logo D. Afonso Sanches (Escolaridade)

Se algum ficheiro não existir, o site continua a funcionar: o botão do CV dá 404 e os logos mostram o ícone por defeito.
